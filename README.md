# Vue Todo 

A simple vue todo list (CRUD) app using [vue 2](https://v2.vuejs.org/v2/guide/) and [vuetify](https://v2.vuetifyjs.com/en/)


### Usage
```
git clone git@github.com:brysonbw/vue-todo.git

cd vue-todo

rm -rf .git 

npm install
```

### Developing
```
npm run dev
```

### Build
```
npm run build
```

### Unit Test
```
npm run test:unit
```

### Lint
```
npm run lint
```
